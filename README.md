### Не проходим проверку [RKNHardering](https://github.com/xtclovver/RKNHardering) без рута, ~~тихо, не спеша, без суеты~~
<hr>

Итак, что же я хочу сказать:

***1. Скрыть наличие TUN-интерфейса и системного прокси на Android без root-прав архитектурно невозможно. Однако можно попытаться минимизировать утечку информации о подключении, например IP адреса сервера, что и предлагает данная инструкция.***

***2. Для пользователей роутеров Keenetic / Netcraze:***
Вы можете установить [SKeen](https://github.com/jinndi/SKeen) непосредственно на роутер. Готовые примеры настроек находятся в папке `examples` (файлы `config-cheburnet*.json`).

Особенности:

- Скрытность: Через Wi-Fi TUN-интерфейс не виден на клиентских устройствах.

- Ограничение: Данный метод не обеспечивает защиту от CDN Pulling.

- Рекомендация: Для прохождения проверок на утечки (CDN Pulling) **на Android-смартфонах лучше попрежднему использовать приложение `sing-box`**. При этом само устройство должно быть подключено к Wi-Fi сегменту роутера, где трафик идет напрямую только через провайдера без резервных подключений.
